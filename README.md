A data analytics dashboard for Fantasy Premier League, built using Python and Streamlit. Pulls live data directly from the official FPL API to provide a league-wide overview of player and team statistics, as well as personalised insights for any FPL manager by entering their Team ID.
Features:

Live player rankings by total points, cost, form, and ownership
Premier League team strength ratings across home and away fixtures
Per-manager squad breakdown with goals, assists, clean sheets, and more
Gameweek points history and cumulative performance charts
Chip usage tracker
